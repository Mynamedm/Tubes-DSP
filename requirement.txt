## Perangkat Lunak (Software)
Python 3.x: Aplikasi ini ditulis dengan Python, jadi pengguna harus menginstal Python 3.x di sistem mereka. Versi 3.6 atau lebih tinggi dianjurkan.

## Library yang Diperlukan:

- OpenCV: Untuk mengakses kamera dan memproses frame video.
Cara install: pip install opencv-python

- NumPy: Untuk manipulasi array dan perhitungan numerik.
Cara install: pip install numpy

- Matplotlib: Untuk visualisasi grafik secara real-time.
Cara install: pip install matplotlib

- SciPy: Untuk pemrosesan sinyal, seperti find_peaks yang ada di kode.
Cara install: pip install scipy
